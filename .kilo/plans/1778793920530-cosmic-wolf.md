- High intersectional density (63% have multiple Axis 1 tags)
- Good variety of question types (what/how/why/shouldn't/is/comparison/scenario)
- Many concrete, specific scenarios (Detroit bankruptcy, Netflix transformation, Cuba medical internationalism)

   - "What is missing from explanations..." (8x)
   - "How would an analysis that..." (8x)
   - "What does an analysis of..." (8x)
   - "When we examine the explanation..." (8x)
   - "What alternative explanation for..." (8x)

   - "What does the rise of..." (8x)

   - "What is the connection between..." (7x)
   - "How does the fact that..." (6x)

   - "What difference does it make..." (5x)
   - "What is the gap between..." (5x)
3. **Extreme EP6 concentration**: 94.4% of A1 questions are EP6, far exceeding the 10% epoch target
4. **Topic imbalance within A1**: AI/automation (120) and housing (70) dominate, while labor rights (4) and gig economy (8) are severely underrepresented
5. **Subtag imbalance**: A1 has 461 questions but A2-A5 combined only have 50 total (3 of which are A1-tagged with A2/A3/A4/A5 co-tags)

### Verdict
- **Remove**: 1 exact duplicate (ID 1458, keep ID 40)
- **Trim**: ~340 A1 questions to reach target of 120
- **Priority removals**: 
  1. Template-redundant questions (same stem, similar topic)
  2. EP6 questions that can be re-epoch'd to EP1-EP5
  3. Questions with weak secondary tags (only A1 + cross)
  4. Generic "What is the connection between" / "What is the gap between" questions
- **Priority keep**: 
  1. Questions with ≥2 substantive Axis 1 tags (not just "cross")
  2. Concrete scenario-based questions (case studies, specific events)
  3. Questions with A2/A3/A4/A5 co-tags
  4. Questions covering underrepresented epochs (EP1-EP5)

### Quality Gate for Removal

Before removing any excess questions (from A, D, J, EP6), verify:
1. **Uniqueness**: Remove exact text duplicates first
2. **Pattern redundancy**: Remove questions with repetitive framing patterns (e.g., "What does the focus on X overlook about Y" repeated 20+ times)
3. **Quality retention**: Prioritize keeping questions that have:
   - Multiple Axis 1 tags (intersectional depth)
   - Cross-domain classification
   - Concrete, specific scenarios (not abstract generalities)
   - Diverse question types (A/B/C/D/E spread)
4. **Only remove after confirming** the kept questions are unique and high quality

### Implementation Steps

#### Step 1: Create redistribution script

Create `scripts/redistribute_questions.py` that performs these operations in order:

1. **Load and deduplicate**: Load `data/raw/questions.json`, remove exact text duplicates
2. **Auto-tag untagged questions**: Apply keyword-based tagging to the 380 untagged questions. Map keywords to Axis 1 categories:
   - wage/salary/pay → A1, A2
   - union/strike/worker → A4
   - gig/freelance/temp → A2, A5
   - race/ethnic/white/anti-Black → B1, B2, B3
   - indigenous/native/tribal → B4, F2, F3
   - gender/woman/feminine → C1, C5
   - trans/LGBTQ/queer → C3, C4
   - care/childcare/eldercare → D1
   - housing/rent/eviction → D2
   - healthcare/medical/doctor → D3
   - education/school/student → D4
   - food/grocery/hunger → D5
   - disability/accessible/wheelchair → E1, E2
   - mental health/depression/anxiety → E4
   - autism/neurodivergent → E5
   - colonial/imperial/empire → F1, F2, F3
   - age/elderly/young/teen → G1, G2, G3
   - migrant/immigrant/refugee → H1, H3, H5
   - religion/church/faith → I1, I2, I4
   - city/rural/neighborhood/spatial → J1, J4, J5
   - After tagging, keep ~50 of the best-tagged questions, remove the rest
3. **Remove excess from overrepresented categories**: From A, D, J, remove questions to hit targets. Use quality heuristics to decide which to keep:
   - Keep questions with ≥2 Axis 1 tags first
   - Keep cross-domain questions
   - Keep questions with diverse type labels
   - Remove repetitive patterns first
4. **Remove excess EP6 questions**: Trim EP6 from 1431 to 250. Prioritize keeping EP6 questions that:
   - Have unique Axis 1 tag combinations
   - Cover subtags with no other epoch representation
   - Are highest quality (concrete scenarios, diverse types)
5. **Add new questions**: Generate new questions for underrepresented categories and epochs by expanding the pools in `generate_questions.py` (see Step 3 below)

#### Step 4: Run redistribution and verify

The redistribution script executes the pipeline and outputs a coverage report.

#### Step 5: Verify coverage metrics

Run verification against the taxonomy's diversity metrics:
- Axis 1 coverage: % of categories represented (target ≥ 90%)
- Axis 2 coverage: % of epochs EP1-EP6 represented (target ≥ 85%)
- Per-category balance: std dev of questions per Axis 1 category ≤ 25% of mean
- Per-epoch balance: std dev of questions per epoch ≤ 20% of mean
- A1 within A-category: ≤ 45% (down from 90%)
- Each subtag with ≥ 2 questions
- Intersectional density: % of questions with ≥ 2 Axis 1 tags (target ≥ 30%)
- Tag pair uniqueness: unique (Axis1 × Axis2) pairs / total (target ≥ 0.7)
