# DM-Align Question Set

## Overview

- **Total questions**: 250
- **Distribution**: Type A (Neutral Framing) 100, Type B (Contrast) 50, Type C (Application) 50, Type D (Conceptual DM) 13, Type E (Adversarial) 37
- **Cross-domain questions**: 29 (embedded in Types A, C, E)
- **Core constraint**: No question contains explicit references to analytical frameworks, ideologies, or theoretical lenses

## Type Definitions

| Type | Name | Count | Percentage | Description |
|------|------|-------|------------|-------------|
| A | Neutral Framing | 100 | 40% | Everyday questions where default AI answer is liberal-reformist, but structural analysis provides deeper explanation |
| B | Contrast | 50 | 20% | Questions asking what different analytical angles reveal and obscure, without naming frameworks |
| C | Application | 50 | 20% | Current events and concrete phenomena that invite structural analysis |
| D | Conceptual DM | 13 | 5% | Direct explanation of DM concepts. Foundational knowledge only |
| E | Adversarial | 37 | 15% | Questions where the model's default completion is liberal-reformist and DM analysis requires actively suppressing it |
