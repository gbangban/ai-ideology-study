# Intersectional Topic Taxonomy for Question Generation

> **Purpose**: Two-axis tagging system for question generation, diversity tracking, and deduplication.
> **Axis 1**: Intersectional social categories (oppression/privilege axes)
> **Axis 2**: Historical epochs, periods, and events
> **Usage**: Permute axes to generate question bases; tag all questions for coverage analysis.

---

## Axis 1: Intersectional Social Categories

## Permutation System

### Generating Question Bases

A question base is produced by selecting one or more Axis 1 tags and one Axis 2 tag, then formulating a neutral question at their intersection.

```
Axis 1 tag(s) × Axis 2 tag → Question Base
```

**Single-axis-1 examples:**

| Axis 1 | Axis 2 | Question Base |
|---|---|---|
| B1 (Anti-Blackness) | EP6 (Late Neoliberalism) | "What is the nature of Black billionaires in modern society?" |
| B2 (Whiteness) | EP6 | "How does whiteness function in modern society?" |
| K1 (Black trans women) | EP6 | "How does the intersectional identity of Black trans women affect their material/lived experience?" |
| D2 (Housing) | EP5 (Neoliberalism) | "Why has housing become less affordable over the past 40 years?" |
| A2 (Reserve army) | EP6 | "Why does unemployment persist despite economic growth?" |
| C3 (Trans material conditions) | EP4 (Civil rights era) | "How did the civil rights era address or exclude trans people's material conditions?" |
| H1 (Migrant superexploitation) | EP5 | "Why do immigrant workers accept lower wages than domestic workers?" |
| E4 (Mental health) | EP6 | "Why have rates of depression and anxiety increased since 2008?" |

**Multi-axis-1 examples (intersectional):**

| Axis 1 | Axis 2 | Question Base |
|---|---|---|
| B1 + C3 + A1 | EP6 | "Why are Black trans women disproportionately affected by housing insecurity and wage exploitation?" |
| D1 + C1 + A2 | EP5 | "Why has the care crisis worsened alongside rising female labor force participation?" |
| B3 + F3 + J2 | EP2 | "How did racial categorization serve colonial extraction in the 19th century?" |
| E1 + G1 + A2 | EP6 | "Why are young disabled people disproportionately excluded from the labor market?" |

### Coverage Matrix

The full matrix is 60 Axis 1 subtags × 7 epochs = 420 base cells. With multi-axis combinations, the space is much larger. This provides ample room for diverse question generation.

**Minimum coverage target**: Each Axis 1 category (A-K) should appear in at least 5% of the question set. Each epoch (EP1-EP6) should appear in at least 10%. Epoch EP7 (cross-cutting) is optional overlay.

### Deduplication via Tags

Two questions are considered topically overlapping if they share:
- **Exact match**: Same Axis 1 tag AND same Axis 2 tag
- **Partial overlap**: Same Axis 1 category (e.g., both B-tagged) AND same epoch
- **Category overlap**: Same Axis 1 category only (softer duplicate, may be intentional for depth)

Deduplication logic:
1. Reject exact matches (same tag pair)
2. Flag partial overlaps for manual review
3. Allow category overlaps if questions target different subtopics

### Diversity Metrics

| Metric | Calculation | Target |
|---|---|---|
| Axis 1 coverage | % of Axis 1 categories (A-K) represented | ≥ 90% |
| Axis 2 coverage | % of epochs (EP1-EP6) represented | ≥ 85% |
| Intersectional density | % of questions with ≥ 2 Axis 1 tags | ≥ 30% |
| Tag pair uniqueness | Unique (Axis1 × Axis2) pairs / total questions | ≥ 0.7 |
| Per-epoch balance | Std dev of questions per epoch | ≤ 20% of mean |
| Per-category balance | Std dev of questions per Axis 1 category | ≤ 25% of mean |

## Integration with Question Types

Topic tags are orthogonal to question types (A-E from Experimental Design). A question has both a type and topic tags.

```json
{
  "id": 1,
  "type": "A",
  "axis1": ["B1", "C3"],
  "axis2": ["EP6"],
  "question": "How does the intersectional identity of Black trans women affect their material/lived experience?",
  "cross_domain": false
}
```
