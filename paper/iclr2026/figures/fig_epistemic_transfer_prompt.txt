IDEOGRAM4 COMFYUI WORKFLOW — Epistemic Transfer Diagram
========================================================

How to use:
1. Open your Ideo4.json workflow in ComfyUI
2. Find the "Ideogram4PromptBuilderKJ" node (node id 185)
3. Copy the JSON content from fig_epistemic_transfer_ideogram4.json
4. Paste it into the "import_json" input field of the Ideogram4PromptBuilderKJ node
   (or use the node's import button if available)
5. Queue prompt

The JSON maps to these Ideogram4 fields:

  high_level_description — Overall scene description for the model
  background — Pure white, clean canvas
  style — "3d" (best for technical diagrams in Ideogram4)
  style.photo — Detailed style constraints (modern minimal, rounded shapes, etc.)
  aesthetics — Professional academic aesthetic
  lighting — Flat, even (no dramatic lighting that would distort the diagram)
  medium — Digital illustration
  bboxes — 9 region descriptors that guide composition:
    - 3 main stage boxes (Pre-trained, SFT, Finetuned)
    - 2 connecting arrows
    - 1 dashed feedback loop arrow
    - 2 bottom outcome boxes (Formal Logic green, Applied ID coral)
    - 1 divider text label

Each bbox has:
  x, y — Normalized position (0-1)
  w, h — Normalized width/height (0-1)
  type — "obj" for visual elements, "text" for text regions
  desc — Detailed description of what should appear in that region
  text — (for type="text") exact text to render

Resolution: 1920x1080 (landscape, fits full-width ICLR figure)
